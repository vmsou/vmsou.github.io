Para esse projeto, tivemos a ideia de desenvolver um jogo baseado no Universo de Attack on Titan, porém com uma história original, o jogo segue uma plataforma 2D, o jogador tem acesso a equipamento de manobra bidimensional.
Enquanto explora uma caverna, há de se defender de inimigos colossais, também conhecidos como Titãs, isso enquanto tem que se preocupar com a munição de seu equipamento recarregando o gás que o propulsiona.

Para a criação de um jogo através da Platarforma Processing. Nossa equipe decidiu fazer um que combine com a criação geométrica fornecida pelo processing, então assim, decidimos tentar recriar o jogo Atari Breakout, em que diversas funcionalidades foram adicionadas, como a criação dinamica dos elementos do jogo, como as bolas e os retângulos, assim como a opção de salvar o jogo, configurações como volume e mapeamento dos controles, e o placar através de JSON e um algoritmo de ordenação.

Para a realização dessa atividade, era exigido que tivessem alguns recursos. Sendo estes imagens, videos e inputs. Para a parte da imagem nossa equipe decidiu criar um jogo da memória com a utilização de cartas do Yu-Gi-Oh! e Pokemon. Para a parte das imagens foi decidido um lugar que reunisse os trailers lançados em 2021 através de um arquivo JSON e para input um sistema de cadastramento com base de dados através do XAMPP, parte realizado com JavaScript e a outra com PHP.
